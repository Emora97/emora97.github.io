About

This repository contains the source code for my GitHub Pages portfolio website. The site highlights selected engineering projects, research activities, technical skills, and supporting documentation developed throughout my undergraduate studies in Mechanical Engineering at the Illinois Institute of Technology.

The portfolio currently features projects involving:

* Autonomous microdrone control using the Bitcraze Crazyflie 2.1+
* Computer vision–based trajectory tracking and validation with OpenCV
* Inertial measurement unit (IMU) stabilization systems
* Robotics and mechatronics development
* Experimental design and data visualization
